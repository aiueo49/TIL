# GitHub CLI
RUNTEQカリキュラムのおかげですでにインストールはされていた。
あとはコマンド操作を覚えるだけ。

# 操作

### issueを立てる

```
gh issue create
```

### オープンしているissueを確認する

```
gh issue list
```

### issue1の詳細を見る

```
gh issue view 1
```
