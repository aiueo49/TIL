## 経緯
Runteq Report 作成ツールの通信が304だったので何だろうと気になったので調べてみた。
200が成功。400と500が失敗ということぐらいしか知識としてなかった。

## 結論
304　はエラーではなかった。

> 「新しくサーバからデータを貰わないで自分が前にダウンロードして表示したデータを使ってね！」
ブラウザ内のキャッシュに残っているコンテンツを使って表示させるときなどにこれが帰ってきています。
